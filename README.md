# Simple Web Application

Simple web application using embedded tomcat and downloaded/unzipped (not installed) JDK (OpenJDK).

### Usage

Clone the project or download a zip release and run the scripts accordingly:
- On Windows: `run.bat`
- On Linux: `run.sh`
